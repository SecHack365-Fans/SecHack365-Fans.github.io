# location_for_what
## 問題文
とある場所を友達と探索していると、「ここ、何かの映画の聖地だった気がするけど、名前忘れちゃった......」とのこと。
シュッと特定して教えてあげよう！

Flagの形式は `TsukuCTF23{映画のタイトル}` 

## 難易度
Easy

## 解法
0. Google Lensで新宿御苑内の東屋であることが分かる
1. "新宿御苑 映画 聖地" でググると、"言の葉の庭" の聖地であることが分かる
